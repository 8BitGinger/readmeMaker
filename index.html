ReadMe Maker

This is a simple App using Node.js to create a high quality ReadMe file for Projects

How to Use!

  • 1 - Start by Cloning the Repo
    • VIEW REPO
  • Open the Repo using VS Code
  • Right-Click the file
    index.js
    and Open in Integrated Terminal
  • Type the following in the terminal:
    node index.js
    and press Enter
  • Follow the Prompts
  • Your personalized ReadMe will be saved as:
    newREADME.md
